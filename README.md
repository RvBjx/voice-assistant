# voice assistant
Made with python, wake word and multiple commands

## Libraries:
- pvporcupine
- struct
- pyaudio
- speech_recognition
- pyttsx3
- webbrowser
- random
- requests
- bs4 (Beautiful Soop)
- logging
- time
- phue
- json
- pyjokes
- os
- keyboard

## Commands:
- hello
- goodbye
- search in google
- search in youtube
- play music on youtube
- weather for pre-defined location
- philips hue lights on / off
- tell a joke
- change user's name
